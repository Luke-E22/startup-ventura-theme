# Startup Ventura

Website for **Startup Ventura**, a 501(c)(3) nonprofit startup accelerator in Ventura County, California.

This is a **static site**, generated by a Node script and deployed to **Netlify**. There is no CMS and no backend (forms use Netlify Forms; donations use a Zeffy embed).

## Where things live

| Path | What it is | Edit here for... |
|------|-----------|------------------|
| `preview/gen.mjs` | The site generator — every page, all copy, SEO/meta, JSON-LD, sitemap, robots.txt, llms.txt | Content, pages, structure, SEO |
| `startup-ventura/assets/css/main.css` | All styles | Design / layout |
| `startup-ventura/assets/js/main.js` | All client JS (nav, carousel, forms, modals) | Behavior |
| `startup-ventura/assets/img/`, `fonts/` | Images and self-hosted fonts | Media |
| `preview/build-dist.mjs` | Packages everything into `dist/` (Netlify's build command); also minifies the CSS | Build/deploy config |
| `scripts/optimize-images.mjs` | Regenerates WebP + resized image variants with `sharp` (run locally) | After adding/replacing photos |
| `scripts/fetch-events.mjs` | Pulls the /events schedule from the Notion "Website Events" DB at build time (fail-soft to `preview/events-data.json`); nightly rebuild via `.github/workflows/nightly-rebuild.yml` | Events: edit the NOTION database, not code |
| `scripts/og-card.html` | Source for the social share card (rendered to `assets/img/og/og-default.jpg`) | The link-preview image |
| `netlify.toml` | Build command, pretty-route redirects, cache headers | Routing / caching |

> The `startup-ventura/` folder name is legacy (it was once a WordPress theme). Only `startup-ventura/assets/` remains and is load-bearing — the build copies it to `dist/assets/`.

## Build & preview

```bash
node preview/build-dist.mjs   # -> dist/ (the deployable site; also writes preview/site/)
```

Local preview is served from `preview/site/` (regenerated on every build; git-ignored).

## Deploy (Netlify)

Netlify runs `node preview/build-dist.mjs` and publishes `dist/` (both set in `netlify.toml`). Live domain: `startupventura.com`.

## Notes

- **Forms** are Netlify Forms (contact, notify, newsletter, partner, careers). Enable *Form detection* in the Netlify dashboard and add notification routes.
- **Give** opens a Zeffy donation modal; it only works on Zeffy's allowlisted origins (the live domain), not on localhost.
- **Analytics**: set `ANALYTICS_ID` in `gen.mjs` to a GA4 `G-…` id (or GTM container) to turn on tracking site-wide; `/thank-you` fires the donation conversion.
